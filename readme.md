git init → Initializes a new Git repository in the project

git add . → Adds all files to the staging area

git commit -m "message" → Saves the changes with a commit message

git branch --list → Displays the list of all branches

git branch devlopment → Creates a new branch named devlopment

git checkout devlopment → Switches to the devlopment branch

git push origin master → Pushes the code to the master branch on GitHub
